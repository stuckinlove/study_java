### 함수형 프로그래밍의 조건.
first class citizen에 만족해야함.(First Class Function을 지원하는 언어라고 말함.)<br>
1. 함수를 메소드 파라미터로 넘길 수 있어야 한다.
2. 함수를 메소드의 결과값으로 받을 수 있어야 한다.
3. 데이터 구조. 예를 들어 리스트 안에 함수를 넣을 수 있어야 한다. (저장 또는 viriable에 assign할 수 있어야 한다.)
